SCORPIO platform controller
============================

### Based on STM8

PINS description

* D6 - USART Rx
* D5 - USART Tx
* D0..D2 - select motors pair
* D3, D4 - select motor from pair
* C1..C3 - LED1..LED3
* B0..B3 - stepper motors' outputs
* B4, B5, F4 - relays (shutter, neon, flat)
* A1, A2 - end-switches ("-" and "+")
